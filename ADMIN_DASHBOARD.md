## 🚀 دليل الاستخدام

### تسجيل الدخول للإدارة
```
1. ادخل /admin-login
2. ادخل بيانات حساب الأدمن
3. ستُعاد التوجيه إلى /admin/dashboard
```

### الملاحة بين الصفحات
- من AdminDashboard: استخدم الروابط السريعة
- أو استخدم الـ URL مباشرة:
  - `/admin/reports` — مراجعة التقارير
  - `/admin/tenants` — إدارة الشركات
  - `/admin/subscriptions` — الاشتراكات
  - `/admin/users` — المستخدمون
  - `/admin/companies` — الشركات المُقيّمة

### العمليات الشائعة

**الموافقة على تقرير:**
1. ادخل `/admin/reports`
2. اختر تقرير من القائمة
3. اضغط "موافقة"
4. التقرير سيُرفع و سيُحدّث مؤشر الثقة تلقائياً

**تجديد اشتراك منتهي:**
1. ادخل `/admin/subscriptions`
2. اضغط "تجديد" على الاشتراك المنتهي
3. الاشتراك سيُجدّد لمدة 90 يوم إضافي

**إيقاف حساب شركة:**
1. ادخل `/admin/tenants`
2. اضغط "علق" على الشركة المطلوبة
3. الحساب سيُعلّق و لن تستطيع الشركة رفع تقارير
