## 🏢 **لوحة تحكم الشركات (Company Dashboard)**

### **11 صفحة — للشركات والمساهمين**

#### 📍 **الوظيفة الأساسية:**
- ✅ البحث والتحليل والتقارير
- ✅ المساهمة والتقييم
- ✅ مراقبة الشركات الأخرى
- ✅ إدارة الاشتراك والبيانات الشخصية

#### 📄 **الـ 11 صفحات:**

```
1️⃣  لوحة التحكم (Dashboard)
    └─ 4 KPIs: بحث متبقي، تقارير معتمدة، قوائس مراقبة، مساهمات
    └─ نشاط حديث
    └─ بطاقة "Give to Get"

2️⃣  البحث عن الشركات (Search)
    └─ حقل بحث متقدم
    └─ 4 فلاتر: قطاع، مدينة، مستوى مخاطر، مؤشر ثقة
    └─ عرض النتائج بطاقات

3️⃣  إضافة شركة للسجل (Add Company)
    └─ نموذج: الاسم، السجل، القطاع، المدينة
    └─ طلب موافقة من الإدارة

4️⃣  إضافة تقرير - معالج 4 خطوات (Add Report) ⭐
    └─ الخطوة 1: اختيار الشركة
    └─ الخطوة 2: تفاصيل التعامل
    └─ الخطوة 3: المراجعة
    └─ الخطوة 4: الإقرار والإرسال

5️⃣  تقاريري المرسلة (My Reports)
    └─ جدول التقارير المرسلة
    └─ الحالات: معتمد، قيد المراجعة، مرفوض

6️⃣  تقرير الشركة الشامل (Trust Report) ⭐
    └─ مؤشر الثقة (العداد الدائري)
    └─ 4 حالات عرض:
       • موثوق (≥5 تقارير)
       • أولي (2-4 تقارير)
       • بيانات غير كافية (0-1)
       • مقفل (باقة مجانية)
    └─ إحصائيات شاملة

7️⃣  قوائس المراقبة (Watchlist)
    └─ إضافة شركات للمراقبة
    └─ تنبيهات عند تغير المؤشر

8️⃣  مقارنة الشركات (Compare)
    └─ اختيار حتى 3 شركات
    └─ جدول مقارنة شامل

9️⃣  إدارة الفريق (Team Management)
    └─ دعوة موظفين
    └─ تعيين الأدوار: عرض فقط، موظف، مدير
    └─ تعطيل/حذف الموظفين

🔟 الاشتراك والفواتير (Subscription)
    └─ عرض الباقة الحالية
    └─ ترقية/تخفيض الباقة
    └─ سجل الفواتير

1️⃣1️⃣ الملف الشخصي (Profile)
    └─ تعديل بيانات الشركة
    └─ إعدادات الإشعارات
```

## 🔐 **لوحة تحكم الإدارة (Admin Dashboard)**

### **7 صفحات — لمسؤولي المنصة فقط**

#### 📍 **الوظيفة الأساسية:**
- ✅ مراجعة وموافقة الطلبات
- ✅ تحقق من التقارير المرفوعة
- ✅ إدارة الشركات والمستخدمين
- ✅ رفع بيانات بكميات كبيرة
- ✅ سجل شامل لكل العمليات

#### 📄 **الـ 7 صفحات:**

```
1️⃣  لوحة الإدارة (Admin Dashboard)
    └─ 4 KPIs: طلبات معلقة، تقارير قيد المراجعة، إجمالي الشركات، مستخدمون نشطون
    └─ قائمة المهام المعلقة

2️⃣  مراجعة طلبات الشركات الجديدة (Review Company Requests)
    └─ جدول الطلبات المعلقة
    └─ أزرار: ✓ موافقة / ✕ رفض
    └─ إرسال رسالة تحقق إلى الشركة

3️⃣  مراجعة التقارير (Review Reports)
    └─ جدول التقارير المرسلة من الشركات
    └─ الحالات: قيد المراجعة، معتمد، مرفوض
    └─ أزرار: اعتماد / رفض / طلب استكمال
    └─ كتابة ملاحظات وأسباب الرفض

4️⃣  رفع دفعة شركات (Bulk Import Companies)
    └─ Drag & Drop لملف CSV
    └─ معاينة البيانات قبل الرفع
    └─ رفع آلي بدون مراجعة يدوية

5️⃣  إدارة الشركات (Manage Companies)
    └─ جدول بكل الشركات
    └─ البحث والفلترة
    └─ تعديل: الحالة، الحد الأدنى للتقارير
    └─ تعليق/تفعيل الحسابات

6️⃣  إدارة المستخدمين (User Management)
    └─ جدول المستخدمين مع أدوارهم
    └─ جدول الاشتراكات والباقات
    └─ تعديل الاشتراك يدوياً
    └─ تعطيل/حذف حسابات

7️⃣  سجل العمليات (Audit Logs)
    └─ سجل شامل لكل العمليات
    └─ من فعل، ماذا فعل، متى
    └─ فلترة بـ: نوع العملية، التاريخ، الفاعل
    └─ غير قابل للتعديل (Security)
```

## 📱 **مثال عملي — سيناريو حقيقي**

### **السيناريو:**
شركة "نجد للمقاولات" تريد إضافة تقرير عن شركة "الخليج للتجارة"

### **الخطوات في لوحة الشركات:**
```
1. تسجيل الدخول → الشركة: نجد للمقاولات
2. البحث عن الشركة → "الخليج للتجارة"
3. عرض التقرير → مؤشر الثقة 65/100 (أولي)
4. النقر "إضافة تقرير" → معالج 4 خطوات
5. الخطوة 1: اختيار الشركة ✓
6. الخطوة 2: إدخال البيانات (200,000 ر.س، سداد ممتاز)
7. الخطوة 3: المراجعة ✓
8. الخطوة 4: الموافقة والإرسال
9. رسالة النجاح: "تم الإرسال للمراجعة"
10. المسؤول سيراه في: لوحة الإدارة → مراجعة التقارير
```

### **ماذا يحدث في لوحة الإدارة:**

**المسؤول يرى:**
```
1. تسجيل الدخول → المسؤول (admin)
2. لوحة الإدارة → KPI "التقارير قيد المراجعة: 28"
3. الذهاب لـ "مراجعة التقارير"
4. البحث عن التقرير → من: نجد للمقاولات، عن: الخليج
5. عرض التفاصيل الكاملة + المستندات
6. بدء المراجعة:
   - ✓ اعتماد (يرتفع مؤشر الثقة)
   - ✕ رفض (مع كتابة السبب)
   - ⚠️ طلب استكمال
7. تسجيل في سجل العمليات
```

## 🔐 **أمان البيانات — Row-Level Security (RLS)**

### **لوحة الشركات:**
- ✅ الشركة ترى **فقط** بيانات الشركات الأخرى
- ✅ **لا تستطيع** رؤية بيانات شركات منافسة (إلا الإحصائيات العامة)
- ✅ البيانات مشفرة ومعزولة

### **لوحة الإدارة:**
- ✅ ترى **كل البيانات** من جميع الشركات
- ✅ لكن محدودة بـ **Audit Log** (كل حركة تُسجل)
- ✅ بدون صلاحية تعديل مباشر (موافقة فقط)

## 🎯 **ملخص سريع — تذكر الفرق**

### **لوحة الشركات = ⭐ "أنا مستخدم عادي"**

- أنا أراجع طلبات الشركات الجديدة
- أتحقق من صحة التقارير المرسلة
- أرفع بيانات بالجملة من جهات حكومية
- أدير المستخدمين والاشتراكات
- أحتفظ بسجل شامل لكل حركة
