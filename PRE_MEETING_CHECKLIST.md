# ✅ قائمة التحضير قبل الاجتماع
## Pre-Meeting Preparation Checklist

---

## 📋 قبل الاجتماع بـ 24 ساعة

### من Product Owner:
```
□ تحضير جميع الصور والـ mockups
□ تجهيز جميع المتطلبات الموثقة سابقاً
□ تحديد الحالات الخاصة المعروفة
□ تحضير قائمة الأسئلة
□ مراجعة الـ specification الحالية
□ تحضير جداول المقارنة (خطط الاشتراك)
```

### من Product/Design:
```
□ تجهيز جميع الـ wireframes
□ تحضير user journey maps
□ تجهيز screenshots من النموذج الحالي
□ تحضير شروحات الـ flows
□ تحديد نقاط الالتباس
```

### من فريق التطوير:
```
□ تجهيز قائمة الأسئلة التقنية
□ توثيق التعقيدات المعروفة
□ تحضير نقاط التطلب على الـ backend
□ قائمة بـ assumptions الحالية
```

---

## 🎯 أثناء الاجتماع

### للجميع:
```
□ إيقاف تشغيل الإشعارات
□ إغلاق التطبيقات الأخرى
□ تحضير دفتر ملاحظات
□ الكاميرا والميكروفون يعملان
```

### Facilitator:
```
□ بدء التسجيل (مع الموافقة)
□ مشاركة الشاشة عند الحاجة
□ تتبع الوقت
□ توثيق القرارات بالفعل
□ طلب التأكيدات
```

## 🚨 العلامات التحذيرية

احذر من:

```
⚠️  "سيتم توضيح هذا لاحقاً"
    → اطلب توضيح الآن

⚠️  عدم اتفاق الجميع
    → وثّق الخلاف وأجل الاتفاق

⚠️  "هذا معقد جداً للشرح"
    → قسّم إلى أجزاء أصغر

⚠️  عدم وضوح السلوك
    → ارسم diagram بدلاً من الكلام

⚠️  "نفس الشيء كما هو"
    → اطلب معايير بدلاً من الافتراضات
```

## 🎬 بدء الاجتماع بقوة

### أول 5 دقائق:
```
1. "شكراً لحضوركم"
2. "سنناقش اليوم..." (جدول العمل)
3. "الهدف: توضيح كل شيء لتجنب الالتباس"
4. "سيكون هناك توثيق شامل"
5. "يرجى الانقطاع والسؤال متى لزم الأمر"
```

---

## 📌 بعد كل موضوع

```
□ "هل هناك أسئلة؟"
□ "هل نحن متفقون على هذا؟"
□ "من يوثق هذا القرار؟"
□ "هل هناك حالات خاصة أخرى؟"
→ الانتقال للموضوع التالي
```

---

## 🎯 الخروج من الاجتماع مع

```
✅ وثيقة متطلبات واضحة
✅ رسم جميع الـ flows
✅ قائمة مهام (Tasks) مفصلة
✅ توزيع واضح للعمل
✅ تاريخ اجتماع متابعة
✅ توقيع الجميع على القرارات
```

---

## 💾 التوثيق بعد الاجتماع

### في نفس اليوم:
```
□ تنظيم الملاحظات
□ إنشاء وثيقة المتطلبات النهائية
□ رسم diagrams نظيفة
□ توزيع النتائج على الفريق
□ طلب تأكيد من الجميع
```

---

## 🔍 فحص القائمة النهائية

قبل إنهاء الاجتماع:

```
□ هل تمت مناقشة جميع 19+ صفحة؟
□ هل وضحنا كل الأزرار والـ flows؟
□ هل لدينا إجابات على جميع الأسئلة؟
□ هل الجميع يفهمون؟
□ هل وثقنا كل شيء؟
□ هل تم تقسيم العمل؟
```

إذا كان الجواب "لا" على أي من هذه، يجب توضيح ذلك الآن.
