## 🔍 الميزات المراد تجربتها

### في صفحة Companies
```
1. تحرير حقل البحث → شاهد البيانات تتصفى فوراً
2. اضغط على dropdown الحالة → صفّي حسب النشاط
3. انظر إلى ألوان درجة الثقة:
   - أخضر (> 80) = موثوق
   - أصفر (60-80) = متوسط
   - أحمر (< 60) = محذر
4. تصفح الـ 50 شركة المختلفة
```

### في صفحة Reports
```
1. انظر إلى التقارير المختلفة
2. لاحظ التقييمات من 1-5 نجوم
3. شاهد التقسيم إلى:
   - تقارير إيجابية (👍)
   - تقارير محايدة (😐)
   - تقارير سلبية (👎)
4. جرب البحث والتصفية
```

### في Admin Dashboard
```
1. شاهد 6 بطاقات إحصائيات:
   - 847 شركة
   - 3247 تقرير
   - 12543 مستخدم
   - 78.5% متوسط الثقة
2. انقر على التبويبات الثلاثة
3. شاهد الرسوم البيانية (placeholder)
4. جرب أزرار الإجراءات
```

### في Admin Users
```
1. شاهد جدول 15+ مستخدم
2. جرب البحث بالاسم أو البريد
3. صفّي حسب الدور
4. صفّي حسب الحالة
5. انقر على تعديل لفتح modal
```

```

```
لاحظ الألوان:
- أخضر = جيد/نجح

- أزرق = معلومات
