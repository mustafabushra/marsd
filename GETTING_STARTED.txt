════════════════════════════════════════════════════════════════════════════════
                         STEP 1: Verify Your System (2 min)
════════════════════════════════════════════════════════════════════════════════

Run these commands to verify everything is ready:

  1. Check Node.js:
     node --version
     Expected: v18.0.0 or higher

  2. Check npm:
     npm --version
     Expected: 8.0.0 or higher

  3. Check disk space:
     You need at least 2 GB free
     On Mac/Linux: df -h
     On Windows: dir C:

  4. Check project exists:
     cd marsd
     ls -la agents/

════════════════════════════════════════════════════════════════════════════════
                     STEP 2: Navigate to Agents (1 min)
════════════════════════════════════════════════════════════════════════════════

  cd marsd/agents

Verify you see these files:
  • core/BaseAgent.ts
  • backend/BackendEngineer.ts
  • frontend/FrontendEngineer.ts
  • security/SecurityEngineer.ts
  • qa/QAEngineer.ts
  • orchestrator/AgentOrchestrator.ts
  • cli/MarsadCLI.ts
  • bootstrap.ts
  • package.json
  • tsconfig.json

════════════════════════════════════════════════════════════════════════════════
                    STEP 3: Install Dependencies (3 min)
════════════════════════════════════════════════════════════════════════════════

  npm install

Expected output:
  added 50 packages

If it fails:
  rm -rf node_modules package-lock.json
  npm cache clean --force
  npm install

════════════════════════════════════════════════════════════════════════════════
                       STEP 4: Build TypeScript (2 min)
════════════════════════════════════════════════════════════════════════════════

  npm run build

Expected output:
  ✓ Successfully built

If it fails:
  Check that all .ts files exist
  Run: npx tsc --version

════════════════════════════════════════════════════════════════════════════════
                        IF SOMETHING GOES WRONG
════════════════════════════════════════════════════════════════════════════════

Error 1: "npm install" fails
  → Run: npm cache clean --force
  → Then: rm -rf node_modules && npm install

Error 2: "npm run build" fails
  → Check Node version: node --version (must be 18+)
  → Run: npm ls typescript
  → Try: npm install

Error 3: "npm run bootstrap" doesn't start
  → Check files exist: ls -la
  → Verify build: npm run build
  → Check errors: npx tsc --noEmit

Error 4: "Cannot find module"
  → Run: npm install
  → Clear cache: npm cache clean --force
  → Reinstall: npm install

For detailed help:
  → Read: TROUBLESHOOTING.md
  → Check: READINESS_CHECKLIST.md
  → Review: agents/README.md
