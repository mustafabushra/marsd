## 📋 البيانات المطلوبة للإطلاق

### **1️⃣ حسابات AWS (ضروري جداً)**
- [ ] AWS Account ID
- [ ] Access Key ID
- [ ] Secret Access Key
- [ ] مفضل Region: `me-south-1` أو `eu-central-1`

**الاستخدام:** رفع قاعدة البيانات والملفات والـ CDN

---

### **2️⃣ بوابة الدفع (ضروري جداً)**
- [ ] اختيار البوابة: **Moyasar** أو **Tap**
- [ ] API Key
- [ ] Secret Key
- [ ] حساب تاجر (Merchant Account)

**الاستخدام:** معالجة اشتراكات المستخدمين

---

### **3️⃣ البيانات المالية**
- [ ] أسعار الباقات النهائية:
  - [ ] Free (0 ريال)
  - [ ] Basic (~299 ريال/شهر)
  - [ ] Pro (~799 ريال/شهر)
  - [ ] Enterprise (تسعير مخصص)

---

### **4️⃣ النطاق والـ DNS (ضروري)**
- [ ] اسم النطاق المستهدف (Domain Name)
  - مثال: `marsad.sa` أو `www.marsad.app`
- [ ] جاهزية DNS Records
- [ ] SSL Certificate (سيتم إعداده عبر Cloudflare)

---

### **5️⃣ البيانات النصية والقانونية**

#### الكتابة المطلوبة:
- [ ] **الرسالة الترحيبية** — الصفحة الرئيسية
- [ ] **نبذة عن المنصة** — صفحة About
- [ ] **شروط الخدمة** (Terms of Service)
- [ ] **سياسة الخصوصية** (Privacy Policy)
- [ ] **أسئلة شائعة** (FAQ) — على الأقل 10 أسئلة

**اللغة:** العربية (RTL)  
**النبرة:** احترافية وواضحة

---

### **6️⃣ المعلومات الإدارية**

#### بيانات الاتصال الأساسية:
- [ ] بريد إلكتروني للدعم: `support@...`
- [ ] رقم الهاتف: `+966...`
- [ ] عنوان المكتب (اختياري)
- [ ] حسابات التواصل الاجتماعي (اختياري)

#### بيانات الفواتير:
- [ ] اسم الشركة (كما يظهر في الفواتير)
- [ ] السجل التجاري
- [ ] الرقم الضريبي
- [ ] عنوان الفاتورة الكامل

---

### **7️⃣ البيانات الأولية (اختياري لكن مفيد)**
- [ ] قائمة الشركات الأولية (CSV format):
  - اسم الشركة
  - السجل التجاري
  - القطاع
  - المدينة
  - موقع الويب

**الفائدة:** تعبئة قاعدة البيانات الأولية لـ demo
