## 🔄 User Flows المطلوبة توضيحها

### Flow 1: التسجيل → الدفع → الاستخدام
```
1. الزائر يزور الموقع
2. ينقر "ابدأ الآن"
3. يملأ نموذج التسجيل
4. اختيار خطة الاشتراك
5. الدفع (Moyasar/Tap)
6. تفعيل البريد الإلكتروني
7. الدخول للـ Dashboard
8. البدء بالاستخدام

❓ نقاط التوضيح:
  • ما الذي يحدث إذا أرغم المستخدم عملية الدفع؟
  • هل هناك رابط "تجاهل" أو "لاحقاً"؟
  • ما السلوك إذا لم يتم التحقق من البريد؟
```

### Flow 2: البحث → إضافة تقرير → الاعتماد
```
1. المستخدم يبحث عن شركة
2. يختار شركة من النتائج
3. يعرض ملف الشركة وسجل التقارير
4. ينقر "إضافة تقرير"
5. يملأ نموذج 4 خطوات
6. يرفع مستندات
7. يؤكد ويرسل
8. ينتظر اعتماد الإدارة
9. يتلقى إشعار الاعتماد أو الرفض
10. إذا تم الرفض، يمكنه إعادة الإرسال مرة واحدة

❓ نقاط التوضيح:
  • ما الوقت المتوقع للاعتماد؟
  • هل يتلقى بريد تأكيد الإرسال؟
  • كيف يتم عرض حالة التقرير المعلقة؟
```

### Flow 3: المراقبة والإشعارات
```
1. المستخدم يضيف شركة للقائمة
2. النظام يراقب درجة الثقة
3. عند تغيير كبير، يرسل إشعار
4. المستخدم ينقر الإشعار
5. يُعاد توجيهه لملف الشركة المحدث

❓ نقاط التوضيح:
  • ما حد التغيير المطلوب (5٪، 10٪، إلخ)؟
  • كم مرة يمكن إرسال إشعارات يومية (Max)؟
  • هل يمكن تخصيص الإشعارات لكل شركة؟
```

### Flow 4: إدارة المشتركين (Admin)
```
1. المشرف يدخل لوحة التحكم
2. يراجع صف التقارير المعلقة
3. يفتح تقرير واحد
4. يراجع التفاصيل والمستندات
5. يقرر الموافقة أو الرفض
6. إذا رفض، يكتب سبب
7. التقرير يُعاد للمرسل للمراجعة
8. درجة الثقة تُحدّث فوراً

❓ نقاط التوضيح:
  • هل يمكن الموافقة على عدة تقارير دفعة واحدة؟
  • هل هناك انتقاء تلقائي للتقرير التالي؟
  • هل يتم إرسال إشعار فوري للمرسل؟
```

## 📋 المتطلبات الناقصة

### الحقول المفقودة:

1. **تحديد الأدوار والصلاحيات بدقة**
   - من يمكنه رؤية ماذا؟
   - من يمكنه فعل ماذا؟
   - هل هناك تدرج صلاحيات؟

2. **تحديد رسائل الأخطاء**
   - ماذا يقول النظام إذا فشل الدفع؟
   - ماذا إذا تجاوز حد الخطة؟
   - ماذا إذا انتهت صلاحية الجلسة؟

3. **تحديد الإشعارات**
   - أي الأحداث تُرسل إشعارات؟
   - القنوات (Email, In-app, SMS)؟
   - وقت الإرسال؟

4. **تحديد الحالات الحدية (Edge Cases)**
   - ماذا إذا لم يتم التحقق من البريد؟
   - ماذا إذا انتهى الاشتراك؟
   - ماذا إذا تم تعليق الحساب؟
