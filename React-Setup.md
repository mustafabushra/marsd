## التصميم المعتمد

- **أسلوب:** React Components مع React Router
- **الألوان:** Navy #1E2A52, Green #16A34A, Sky #F8FAFC
- **الخط:** Tajawal من Google Fonts
- **RTL:** دعم كامل للعربية (dir="rtl")
- **State Management:** useState + Context API

## تعليمات التثبيت
