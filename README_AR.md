# مرصد 📊 — منصة تقييم الشركات السعودية

**مرصد** هي منصة SaaS احترافية لتقييم وتتبع موثوقية الشركات السعودية من خلال تقارير المجتمع.

---

## 🎯 الرؤية

تحويل بيانات المجتمع عن الشركات إلى **مؤشر ثقة موثوق** يساعد المشترين، الموردين، والمستثمرين على اتخاذ قرارات مستنيرة.

---

## ✨ الميزات الرئيسية

### 🔍 البحث الذكي
- بحث Full Text Search على 8 حقول (الاسم، السجل، الضريبة، القطاع، إلخ)
- Autocomplete مع اقتراحات فورية
- Fuzzy matching للتسامح مع الأخطاء
- نتائج مرتبة حسب الصلة والثقة

### ⭐ مؤشر الثقة
- درجة من 0-100 بناءً على تقارير المجتمع
- 3 مستويات: Full ✅ / Preliminary ⚠️ / Insufficient ❌
- مخاطر: منخفضة 🟢 / متوسطة 🟡 / عالية 🔴

### 📋 نظام التقارير
- إرسال تقارير عن الشركات (دفع، التزام، إلخ)
- مراجعة من الإدارة
- منع التقارير المكررة (90 يوم)
- نظام نقاط (Give-to-Get)

### 📊 التحليلات
- Timeline للتقارير الحديثة
- اتجاهات الأداء (تحسن/تراجع/مستقر)
- ملخص التقارير بالنوع
- إحصائيات الشركة

### 👥 إدارة المستخدمين
- 4 أدوار: مالك، إداري، مدير، عارض
- 12 صلاحية مُحددة
- قائمة مراقبة شخصية
- إدارة فريق الشركة

### 🛡️ الأمان
- Row-Level Security (RLS)
- Clerk authentication
- حماية هوية المبلغين
- صلاحيات محكمة على كل إجراء

## 🔐 الأدوار والصلاحيات

### Owner (المالك)
```
✅ إضافة/تحرير التقارير
✅ إدارة المستخدمين (كاملة)
✅ تحرير بيانات الشركة
✅ إدارة الاشتراكات
✅ عرض التحليلات
✅ تصدير البيانات
```

### Admin (الإداري)
```
✅ إضافة/تحرير التقارير
✅ إدارة المستخدمين (جزئي)
✅ عرض الاشتراكات
✅ عرض التحليلات
❌ تحرير الشركة
❌ إلغاء الاشتراك
❌ حذف المستخدمين
❌ تغيير الأدوار
```

### Manager (مدير)
```
✅ إضافة/تحرير التقارير
✅ عرض التحليلات
❌ إدارة المستخدمين
❌ تحرير الشركة
❌ إدارة الاشتراكات
```

### Viewer (عارض)
```
✅ عرض التحليلات
❌ إضافة التقارير
❌ إدارة المستخدمين
❌ تحرير أي شيء
```

## 📊 نظام النقاط (Credits)

### كيف يعمل
```
1. عند التسجيل: 100 نقطة مجاني
2. عند إرسال تقرير: -1 نقطة
3. عند قبول تقريرك: +10 نقاط
4. عند تحديث الاشتراك: +نقاط إضافية
```

### الحدود
```
Free: 10 نقاط/شهر
Basic: 100 نقطة/شهر
Pro: 500 نقطة/شهر
Enterprise: غير محدود
```
