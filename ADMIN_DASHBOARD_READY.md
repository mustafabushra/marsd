```
✅ قائمة التقارير المعلقة:

   • يتحول اللون عند الاختيار

✅ لوحة التفاصيل:
   • معلومات الشركة
   • نوع التقرير (مع أيقونة)

   • التاريخ

      • يظهر رسالة نجاح
   
   ❌ زر "الرفض" (أحمر)
      • يحذف التقرير من القائمة
      • ينتقل للتقرير التالي تلقائياً
      • يظهر رسالة الرفض

✅ تصميم احترافي:
   • 2/3 عرض للقائمة، 1/3 للتفاصيل
   • تفاعل سلس عند الاختيار
   • بيئة عمل منظمة وواضحة

✅ رسالة نهائية:
   • عندما تنهي جميع التقارير
   • تظهر رسالة "لا توجد تقارير معلقة! ✅"
   • احتفالي ومحفز
```
